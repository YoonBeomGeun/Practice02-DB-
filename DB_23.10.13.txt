개념정리>
1. 정형데이터란?
- 저장할 데이터(항목)가 정해져있는 데이터
- 비정형 데이터: 저장할 데이터가 정해져 있거나 정해져 있지 않은 데이터

2. DBMS란?
- Database Management System
- DB를 관리해 주는 프로그램

3. 알고있는 DBMS의 종류 2가지 이상
- MySQL, MongoDB, 오라클

4. SQL은 무엇의 약자?
- Structured Query Language(체계적으로 정리된 명령문의 모음)

5. SQL의 종류 3가지 분류
- DDL, DML, DCL

6.CRUD는 무엇의 약자?
- Create, Read, Update, Delete

7. 알고있는 DB클라이언트 프로그램
- DBeaver

8. DML중 데이터를 넣을 때 사용하는 구문
- insert

9. DML중 데이터를 검색할 때 사용하는 구문
- select
